Questionário

Pergunta 1
Na tag <input> com o atributo type="text", para que serve o atributo "placeholder"?
Resposta
O atributo "placeholder" faz você colocar uma escrita na área atribuida ao " type='' ", podesndo usar como identificação em espaços para colocar nome de usuário, senha, entre outros. Funciona para os <input type=""> que pedem escrita, como "text" e "password".
Pergunta 2
Na tag <input> com o atributo type="radio" ou type="checkbox", para que serve o atributo "checked"?
Resposta
Caso queira deixar uma opção previamente marcada no atributo "radio" ou "checkbox", dentro do elemento <input> vc acrescenta o atributo "checked".
Pergunta 3
Na tag <input> para que serve o atributo "value"?
Resposta
O atributo "value" especifica um valor do elemento <input> usado. Possui usos diferentes dependendo do "input type = '' " utilizado:
  • Para "button", "reset", e "submit"
    • Define o texto do botão
  • Para "text", "password" e "hidden"
    • Define o valor padrão do campo onde insere o texto.
  • Para "checkbox", "radio" e "image"
    • Define o valor associado no input. Também será o valor que será enviado na função do atributo "submit"
Pergunta 4
Que atributo deve ser utilizado para agrupar vários botões de radio em um formulário de modo que apenas um deles fique marcado de cada vez?
Resposta
Em um grupo de atributo "radio", eles devem compartilhar o mesmo nome no atributo "name", para serem tratados como grupo.
Pergunta 5
Para que serve a tag <label> Para que serve o atributo "for" nessa tag?
Resposta
A tag <label> serve para rotular vários elementos <input>. Também promove o benefício de ajudar usuários quando possui dificuldade de clicar em regiões muito pequenas de escolha.
Pergunta 6
Na tag <option> para que serve o atributo "selected"?
Resposta
O atributo "selected" é um atributo booleano que quando presente, inforoma que aquela opção está pré-selecionada quando a página carregar.
Pergunta 7
Nas tags de formulário, para que serve o atributo "required" ?
Resposta
O atributo "required" é um atributo booleano que quando presente, especifíca que aquele "input" deve ser preenchido antes de submeter o "form".
Pergunta 8
Na tag <form> para que serve o atributo "action"?
Resposta
O atributo "action" especifica para onde mandar as informações do "form" quando ele for submetido.
Pergunta 9
Na tag <form> para que serve o atributo "method"?
Resposta
O atributo "method" especifica como mandar os dados do "form". Eles podem ser mandados como variáveis de URL (pelo "method="get" ") ou como pós transação de html (pelo "method="post" ").
Pergunta 10
Para que o valor digitado de uma senha não apareça na barra de endereços do navegador, qual deve ser o valor do atributo "method" na tag <form>?
Resposta
Para dados considerados sensíveis e que não desejamos que apareça visível no URL, devemos usar o method="post"
Pergunta 11
Qual o valor deve ser informado no atributo "method" na tag <form> em um formulário que será usado para fazer upload de arquivos?
Resposta
Para upload de arquivos utiliza o method="get".
